# format-text

```
Usage: format-text <text> [options]

Options:
  -h, --help	Show this help menu and exit
  -b		Bold
  -u		Underline
  -i		Invert background and foreground color
  -c <color>	Background color
  -t <color>	Foreground color (text)
  -n		Append newline character

Operands:
  text		The string to be formatted
  color		One of: black|red|green|yellow|blue|magenta|cyan|white
```
